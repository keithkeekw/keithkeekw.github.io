Basic Git Commands for Beginners

git clone

  • The function of the git clone to save a copy of the main branch from Github into the local hard drive.

  • Below is the example on how to use this command:

      git clone https://github.com/keithkeekw/test_repo.git

git status

  • It is to get the status of the working files. If there’s any changes on any files, it will list it accordingly.

  • Below is the example on how to use this command:

    git status

    Below is the example:

git add

  • It is always known as “staging” before submit it to the repository in Github.

  • Below is the example on how to use this command:

    git add .

    📌 If specific a . as shown above, it means that git will add all the files otherwise specify files which required to be added.

    Below is the example:

git commit

  • This command will save all the changes/updates to the local repository.

  • Below is the example on how to use this command:

    git commit -m "Initial commit"

    📌 -m: This option is required. After the option, insert any message in the quotes.

    Below is the example:

git push origin main 1 2

  • In this command, git will submit all the staged changes in the local repository to Github.

  • Below is the example on how to use this command:

    git push origin main

    📌: If use the flag -u, there's no need to specify the origin and main parameters.
