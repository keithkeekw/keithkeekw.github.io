Installation of PostgreSQL in Raspberry Pi

2020-11-02
postgresql raspberry pi

Installation

  1. Run the following command to install the packages:

    
 sudo apt install postgresql -y

  2. When that’s complete, postsgre user account will be automatically created in the user account. Before proceed further give the postgre account a password. Run the command as below:

    
 sudo passwd postgres

  3. Next, change the user account from pi to postgres.:

    
 su postgres

  4. Proceed to login to the database via psql command and you will be able to access to the database without any password since the user postgres does not have any password assigned to it by default. After login into the psql terminal, proceed to run the command below to create password for postgres account.

    
  ALTER USER postgres WITH PASSWORD `Your Own Password`

Configuration for Remote Access

Configuration on postgresql.config need to be altered in order for pgAdmin to access the cluster from a different computer within the network.

  1. Edit the PostgreSQL config file (/etc/postgresql/11/main/postgresql.config). Uncomment listen_addresses and change its value from ‘localhost’ to '*'.

  2. Edit the pg_hba config file (/etc/postgresql/11/main/pg_hba.config) to comment out the line below:

    #host    all     all     127.0.0.1/32    md5
#host    all     all     ::1/128         md5

    Insert the a new line into the file:

    host     all     all     0.0.0.0/0       trust
host     all     all     ::/0            trust

  3. Restart the PostgreSQL service:

    
 sudo systemctl restart postgresql
