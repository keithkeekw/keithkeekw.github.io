Installation of PostgreSQL in Raspberry Pi

2020-11-02
database
postgresql raspberry pi

Installation

  1. Run the following command to install the packages:

    
 sudo apt install postgresql libpq-dev postgresql-client postgresql-client-common -y

  2. When that’s complete, switch to the Postgres user to configure the database:

    
 sudo su postgres

  3. Create a database user:

    
 createuser postgres -P --interactive

Configuration for Remote Access

Configuration on postgresql.config need to be altered in order for pgAdmin to access the cluster from a different computer within the network.

  1. Edit the PostgreSQL config file (/etc/postgresql/11/main/postgresql.config). Uncomment listen_addresses and change its value from localhost to *.
  2. Edit the pg_hba config file (/etc/postgresql/11/main/pg_hba.config) to change 127.0.0.1 to 0.0.0.0/0 for IPv4 and ::1/128 to ::/0 for IPv6.
  3. Restart the PostgreSQL service: 
    
 sudo systemctl restart postgresql
