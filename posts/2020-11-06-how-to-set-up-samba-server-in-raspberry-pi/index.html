How to Set Up Samba Server in Raspberry Pi

2020-11-06
raspberry pi linux

  1. Ensure the Raspberry Pi’s repositories are up-to-date by performing the command below in the terminal:

    
 sudo apt-get update && sudo apt-get upgrade -y

  2. Next, proceed to install all the required packages from the remote repositories for Samba by running the command below:

       
 sudo apt-get install samba samba-common-bin

  3. After the installation has been completed, we can proceed to create a folder to be shared via the Samba Server. For this note as example, proceed to create a folder in the home directory:

       
 mkdir /home/pi/shared

  4. In order to share the newly created folder in the previous step, we need to configure the settings in the smb.conf file.

       
 sudo nano /etc/samba/smb.conf

    📌 All the Samba configuration resides in this file. Edit with care.

  5. At the end of the file, proceed to add the following to the configuration file:

     [mySharedFolder]
 path = /home/pi/shared
 writeable=Yes
 create mask=0777
 directory mask=0777
 public=no

    Run the following command to create the user. You will be prompted afterward to enter the password.

    
    sudo smbpasswd -a pi

  6. As for the final step, proceed to restart the samba service (smbd):

    
 sudo systemctl restart smbd
