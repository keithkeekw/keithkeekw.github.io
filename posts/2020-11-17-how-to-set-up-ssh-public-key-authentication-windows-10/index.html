How to Set Up SSH Public Key Authentication Windows 10

2020-11-17
windows

  1. Launch the Command Prompt and proceed with the command below:

     ssh-keygen

  2. Ignore the passphrase by simply hit the Enter key.

  3. After generating both the private and public keys, proceed to login to the server via SSH using the username & password.

  4. Once you are in the server, proceed to create a folder as shown below:

     sudo mkdir -p /.ssh

  5. Create a file with name of authorized_keys:

     touch ~/.ssh/authorized_keys

  6. Populate the ~/.ssh/authorized_keys file with the public key which was generated in Windows 10.

  7. To ensure that the contents of ~/.ssh/authorized_keys is your public key, run the command below:

     more ~/.ssh/authorized_keys

  8. once everything is completed, you should be able to SSH into the server without any password.
